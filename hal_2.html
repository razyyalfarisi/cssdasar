CSS DASAR

Pengertian CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mengatur tampilan halaman web. HTML hanya menyusun struktur (judul, paragraf, gambar), sedangkan CSS membuat tampilannya lebih indah (warna, ukuran, posisi, layout).

Utama CSS Memberi warna pada teks dan latar belakang Mengatur jenis dan ukuran huruf Mengatur tata letak, jarak antar elemen, dan posisi Membuat desain halaman lebih menarik dan konsisten

Cara Penulisan CSS


Inline CSS → ditulis langsung pada elemen HTML.
Internal CSS → ditulis di dalam file HTML, biasanya di bagian head.
External CSS → ditulis pada file terpisah (style.css) lalu dihubungkan dengan HTML.

Struktur CSS


Setiap aturan CSS terdiri dari selector, property, dan value. Selector menentukan elemen mana yang ingin diatur (misalnya paragraf, judul, dll). Property adalah atribut gaya yang ingin diubah (misalnya warna, ukuran, posisi). Value adalah nilai dari property (misalnya merah, 20 piksel, rata tengah).
Kelebihan CSS
Memisahkan desain dari struktur HTML
Membuat desain lebih mudah diatur dan diubah
Dapat digunakan berulang pada banyak halaman web Membuat tampilan web lebih profesional
