CSS Eksternal

CSS Eksternal adalah cara menuliskan aturan CSS di file terpisah (biasanya bernama style.css) yang kemudian dihubungkan ke file HTML.

ciri ciri css Eksternal


Ditulis di luar file HTML, jadi HTML hanya fokus pada struktur.
File CSS bisa dipakai oleh banyak halaman sekaligus, sehingga lebih efisien.
Perubahan tampilan cukup dilakukan di satu file CSS, otomatis semua halaman web yang terhubung ikut berubah.

Kelebihan css Eksternal

Kode HTML lebih bersih karena tidak bercampur dengan CSS.
Desain lebih konsisten antar halaman.
Mudah dikelola, apalagi jika halaman web jumlahnya banyak.
